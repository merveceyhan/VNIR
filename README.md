# VNIR

Wheat Classification

In this study, it is aimed to classify wheat. Three types of wheat are used to create the model. Other types of wheat will be added after the model is created.

- Go to BugdayOlcum_CSV folder.

- Extract wheat files from rar and place them in the same directory with vnir.py. If you use different folder directory, you should update folder path in vnir.py.
